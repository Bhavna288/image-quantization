This project includes the source code and executable for a basic image processing application.

Included Files:
- Image.cpp: Contains the definitions for the MyImage class methods, including image reading, writing, and modification logic.
- Image.h: The header file for the MyImage class with declarations for image manipulation functions.
- Main.cpp: The main entry point of the application. It handles the initialization and message loop.
- resource.h: A header file with resource identifiers for the application.
- Image.sln: The solution file for Microsoft Visual Studio.
- Image.vcxproj: The Visual C++ project file for Microsoft Visual Studio.

Executable:
- Image.exe: The compiled executable of the application.

To run the application, use the following command:
Image.exe [ImagePath] [ZoomFactor] [RotationAngle] [FramesPerSecond]
